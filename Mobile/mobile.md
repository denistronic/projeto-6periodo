# Arquivo Mobile


Para iniciar o projeto, entrar na pasta sabermais, e em um terminal inicializar o Metro

`npx react-native start`

Em outro terminal iniciar

`npx react-native run-android`

Caso dê algum problema com dependência rodar:

`npm ci` 

ou se tiver algum erro:

`npm install`

Depois disso é só rodar o Metro com o `npx react-native start` e o `npx react-native run-android`
