# Arquivo Mobile

Node v25.1.0
NPM 11.6.2
NVM(Node Version Manager) 0.40.3-1
Ionic 7.2.1


Instalar o Ionic

`npm install -g @ionic/cli`

Instalar Cordova, Native-run Capacitor

`npm install -g cordova`
`npm install -g native-run`
`npm install -g @capacitor/cli`

No diretório sabermais rodar:

`ionic serve`

Informações

O diretório sabermais/src/pages estão as páginas do projeto. O diretório sabermais/src/styles estão o CSS do projeto. O diretório saber/src/script são os Javascript. O sabermais/public/assets são das imagens
